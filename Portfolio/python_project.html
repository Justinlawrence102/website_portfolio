Rock Paper Scissors

Python Computer Visions Game

In my AI class I took in the fall of 2019, I learned the fundamentels of computer vision, but we did not do any program examples. I decided I wanted to also teach myself Python, so combining these two things together I spent a month working through tutorials of Python and a computer vision framework called Open VC.

Then, I used all of the new skills I learned to create a python application that allows the users to play Rock Paper Scissors against a computer. Using Open CV, I allow the user to put up a hand signal and the program will detect if it is a rock, papper, or scissor. The scoring is then done automatically in either a best 2 of 3 or best 3 of 5 match.

To create a program that can idenify a hand in an image, and then figure out what shape it is making, I needed to use a variaty of image procressing techinques. First, I had to get a threshold to try to identify where in the hand was. Then, using various iscolation techiques, I am able to find the contours of the hand. With this data, I parse through all of the contours and can get a relativily accurate estimation of how many fingers are being shown. I translate this number into a guess of what shape the user is making. Because I am taking hundreds of images in the couple of seconds that the hand is being detected, I take an average of all of the guesses made to pick the one that is most likily correct. <
