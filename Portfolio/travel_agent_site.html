Travel Agent Website

Apart of a Software Engineering class, I worked on a team of 5 to create a web app for a local client in Gainesville. I was selected as the Scrum Master for the project, and we are using the agile development process to keep the project organized. I was in charge for keeping track of what progress everyone on the team completed week to week, schedule our meetings outside of class time, and manage our github page to ensure all of the team's code merges togther without any problems.

I was also a full stack developer on the team. While I primarily worked on backend code using JavaScript, I also worked on front end code to design the web page using HTML and CSS. I also worked on several of the wireframes for the website.

In this project, I got experience using the Agile Scrum Development process. We created user stories for each of the tasks and used Pivotal Tracker to keep track of each of the stories. Every couple days we had our scrum meetings to get feedback from the rest of the team. For the final submission, we also docuemnted the entire website, including what each page does and how it works.
